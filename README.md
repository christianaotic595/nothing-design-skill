## ✨ What this is

nothing-design-skill is a Claude Code skill for building UI in the Nothing design language. It helps you shape interfaces with a monochrome look, strong type, and a calm industrial feel.

Use it when you want:

- Clean black and white UI
- Simple layouts with strong spacing
- Sharp typography
- A minimal, device-like look
- Interface ideas that match the Nothing style

### 🧩 Style traits

The Nothing design language usually uses:

- Black, white, and gray tones
- Clear type with strong contrast
- Boxy layouts
- Thin lines and hard edges
- Industrial detail
- Space that feels deliberate
- A neat, technical look

## 🛠️ How it works

You do not need to write code by hand to use the skill in the normal way. You can ask Claude Code to build a UI that follows the Nothing style.

Example requests:

- Make a dashboard in the Nothing design language
- Design a black and white mobile screen
- Create a landing page with sharp typography
- Build a settings page with a minimal industrial look
- Turn this rough idea into a Nothing-style UI

The skill helps guide the output so the design stays consistent.
